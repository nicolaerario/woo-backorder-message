A little plugin to change the front end message for backorder products.

== Description ==

This plugin override the default (and confusional) message shown when a product have backorder setting.

== Installation ==

1. Upload `woo-backorder-message` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

== Usage ==

* Replace the message in the variable with your own.
